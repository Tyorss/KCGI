# 5월 시장 데이터 대시보드

## 실행 방법

`market-dashboard.html` 파일을 브라우저에서 열고 `5월.zip`, 개별 Excel/CSV 파일, 또는 `bloomberg` 폴더를 업로드한다. 페이지는 브라우저 안에서만 파일을 읽으며 서버로 데이터를 전송하지 않는다.

## 업로드 파일 설명

지원 파일은 `WTI.xlsx`, `10년물.xlsx`, `CPI.xlsx`, `SPY.xlsx`, `WIRP.xlsx`, `5Y KOSPI PEBD.xlsx`, `삼성.xlsx`, `하이닉스.xlsx`, `DRAM.xlsx`, `CSI300.xlsx`, `HSTECH.xlsx`, `Star50.xlsx`, `QTUM.xlsx`, `NASA.xlsx`이다. 파일명 인코딩이 깨질 수 있어 파일명, 컬럼명, 상단 메타데이터를 함께 사용해 데이터셋을 식별한다.

## 현재 생성 가능한 그래프

업로드된 실제 데이터에 `Date` 헤더와 필수 컬럼이 있을 때만 차트를 생성한다. 매크로, 한국, 미국, 중국, 산업·테마 5개 섹션의 차트 목록은 `available_charts_summary.csv`에 정리되어 있으며, 업로드 후 페이지에서 생성 가능 여부가 자동 판정된다.

## 추가 필요한 데이터

현재 Bloomberg 파일만으로 만들 수 없는 시가총액, 외국인 수급, CAPEX, IPO 이벤트, SpaceX/Reddit/SPCE 관련 항목은 요청 카드로 표시한다. 정적 요청 목록은 `missing_data_requests.csv`에 포함되어 있고, 페이지 상단의 `누락 데이터 요청서` 버튼으로 현재 상태 기준 CSV를 다시 받을 수 있다.

## 데이터 검증 규칙

각 Excel 파일에서 첫 번째 컬럼 값이 `Date`인 행을 실제 헤더로 사용한다. `PX_LAST`, `PX_VOLUME`, `Price`, P/E 밴드 컬럼을 실제 파일에서 찾아 사용하고, `#N/A Requesting Data`가 포함된 파일은 Invalid 처리한다. 기본 기간은 2026년 5월이며, 사용자가 전체 기간 보기로 전환할 수 있다.

## 목데이터 금지 원칙

샘플 배열, 임의 숫자, 무작위 값, 파일에 없는 데이터로 차트를 만들지 않는다. 데이터가 없거나 필수 컬럼이 없으면 차트를 생성하지 않고 필요한 자료 요청 카드로 표시한다.

## PNG/CSV 다운로드 방법

상단 버튼에서 전체 페이지 PNG, 전체 통합 CSV, 누락 데이터 요청 CSV를 다운로드할 수 있다. 각 섹션에는 섹션 PNG와 섹션 CSV 버튼이 있고, 각 차트에는 차트 PNG 버튼과 원천 데이터 접기/펼치기 표가 있다. 수익률 CSV는 업로드 후 페이지의 전체/섹션 CSV와 원천 데이터 표에서 실제 계산값을 확인할 수 있다.
